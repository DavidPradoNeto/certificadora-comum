Data Manipulation Language (DML)

DML é outra parte importante da linguagem SQL que permite manipular os dados em um banco de dados.

Os principais comandos DML incluem:

  • SELECT:

    O comando SELECT é usado para recuperar dados de uma ou mais tabelas em um banco de dados. Ele permite filtrar os dados, classificá-los e realizar operações de junção. É o comando mais comumente usado em SQL.

                            
                            SELECT coluna1, coluna2
                            FROM nome_tabela
                            WHERE condição;
  • INSERT:

    O comando INSERT é usado para inserir novos registros em uma tabela existente. Podemos inserir valores específicos em cada coluna ou inserir registros a partir de uma consulta SELECT.

                            
                            INSERT INTO nome_tabela (coluna1, coluna2)
                            VALUES (valor1, valor2);
  • UPDATE:

    O comando UPDATE é usado para modificar os dados existentes em uma tabela. Podemos atualizar valores em uma ou mais colunas com base em certas condições.

                            
                            UPDATE nome_tabela
                            SET coluna = novo_valor
                            WHERE condição;
  • DELETE:

    O comando DELETE é usado para excluir registros de uma tabela com base em certas condições. Os registros são removidos permanentemente do banco de dados.

                            
                            DELETE FROM nome_tabela
                            WHERE condição;

Esses são os comandos básicos DML no SQL. Cada um deles desempenha um papel importante na criação, modificação e manipulação de dados em um banco de dados.
