O que são Triggers:

Triggers são objetos especiais em bancos de dados que são acionados automaticamente em resposta a certos eventos, como inserção, atualização ou exclusão de dados.
Os triggers são usados para aplicar regras de negócios, realizar validações ou executar tarefas automatizadas.

Como funcionam os Triggers:

Quando um evento específico ocorre, o trigger associado a esse evento é ativado e executa o conjunto de ações definido nele.

Os eventos comuns que acionam triggers incluem:

  • INSERT: O trigger é acionado quando novos dados são inseridos em uma tabela.
  • UPDATE: O trigger é acionado quando os dados em uma tabela são atualizados.
  • DELETE: O trigger é acionado quando os dados são excluídos de uma tabela.

                        
                            CREATE TRIGGER nome_trigger 
                            AFTER INSERT/UPDATE/DELETE ON nome_tabela
                            FOR EACH ROW
                            BEGIN
                                -- Ações do trigger aqui
                            END;

O trigger é associado a uma tabela específica e é executado após um evento (INSERT, UPDATE ou DELETE). As ações do trigger são definidas dentro do bloco BEGIN-END.
