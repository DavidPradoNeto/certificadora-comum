Data Definition Language (DDL)

DDL é uma parte importante da linguagem SQL que permite criar, alterar e excluir objetos de banco de dados, como tabelas, índices e restrições.

Os principais comandos DDL incluem:

  • CREATE:

    O comando CREATE é usado para criar objetos de banco de dados, como tabelas, índices e visões. Por exemplo, CREATE TABLE é usado para criar uma nova tabela.

                            
                            CREATE TABLE nome_tabela (
                                coluna1 tipo_dado1,
                                coluna2 tipo_dado2,
                                ...
                             );
  • ALTER:

    O comando ALTER permite modificar a estrutura de objetos existentes no banco de dados. Por exemplo, ALTER TABLE é usado para adicionar uma nova coluna a uma tabela existente.

                            
                            ALTER TABLE nome_tabela
                            ADD coluna tipo_dado;
  • DROP:

    O comando DROP é usado para excluir objetos de banco de dados, como tabelas ou índices. Por exemplo, DROP TABLE é usado para excluir uma tabela.

                            
                            DROP TABLE nome_tabela;

Esses são os comandos básicos DDL no SQL. Cada um deles desempenha um papel importante na criação e manipulação de tabelas em um banco de dados.
